# CS 571 Project

Linux CMD-Line Tools

## Installation

When cloning this project, make a directory called filestore within the project directory because that is where it the intermediate files are stored.

## Introduction
- Constructed a framework to streamline command-line tools by creating a Directed Acyclic Graph.
- Operated GoLang to parallelize data processing via the DAG, resulting in significant performance improvements.
